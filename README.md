# react-redux-app

React + Redux simple as hell app boilerplate.

Includes:
- React
- Redux
- Redux-thunk
- React-router 3
- React-redux-router
- Sass styles
- Webpack dev server
- Eslint

## Scripts

### Npm start
Launches development environment.

### Npm run build
Creates dist files for production.

### Npm run test
Lints code with eslint.
